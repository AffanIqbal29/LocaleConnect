# LocaleConnect

Connecting you to the heart of your community. LocaleConnect is a hyper-local marketplace platform designed to empower neighborhood shops and artisans.

## Tech Stack

### Frontend
- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [ShadCN UI](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

### Backend & Services
- **Database**: [Firebase Firestore](https://firebase.google.com/docs/firestore)
- **Authentication**: [Firebase Auth](https://firebase.google.com/docs/auth)
- **AI/GenAI**: [Genkit](https://github.com/firebase/genkit) (Google Gemini 2.5 Flash)

## Features
- **Discovery**: Browse local products and shops in your neighborhood in real-time.
- **AI Assistance**: Magic Write for product and shop descriptions using Google Gemini.
- **Commerce**: Real-time cart management and multi-step checkout simulation.
- **Order Tracking**: Comprehensive order history and status monitoring.
